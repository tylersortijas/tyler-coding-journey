Day 1 - Introduction to the Coding World!

The first day of coding with the Odin Project, I learned about mastering the Command Line. We learned about using the terminal, creating a file using touch, creating a directory using mkdir, finding out your location using ls (list).. modifiers include("S", "h"), and being able to copy files using cp. Of course you can remove the file with rm, however there is another option of copying and pasting with mv

All in all a great first day, and I learned a lot more than what I previously new about the terminal and how it worked.
