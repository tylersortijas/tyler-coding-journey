Tyler's Journey to being a Full Stack Developer!

This is meant to be a place for me to record my progress in coding from my first day to my last day on the Odin Project.

Day 1 - Introduction to the Coding World!

The first day of coding with the Odin Project, I learned about mastering the Command Line. We learned about using the terminal, creating a file using touch, creating a directory using mkdir, finding out your location using ls (list).. modifiers include("S", "h"), and being able to copy files using cp. Of course you can remove the file with rm, however there is another option of copying and pasting with mv.

All in all a great first day, and I learned a lot more than what I previously knew about the terminal and how it worked.

Day 2 - Learning Git, Github, and HTML

Today I learned about Git and GitHub. I finally also got to work on HTML (again) which is exciting! I learned how to create a Repository and push things to GitHub. It was all mainly through the Command Line. Also finally got to work with HTML. Got to learn elements, tags, and how to use them in tandem with each other. Very cool and great feeling to be learning more and more things each day.

Day 3 - Doing first HTML Project

Today I did my very first HTML Project for Odin Project. We put the knowledge of the command line, git, github, and HTML to use. The end product was to make a simple food recipe(s). We had to make 4 pages in total. The index page and the following 3 recipes. It was very fun to do and I can't wait to learn more. We also learned about links, images, and lists as well. We implemented all those into the project. Very very coool!
